performance.note_dct¶

performance.note_dct(x, Ndct, sr)[source]¶

Compute the DCT of a signal and approximate it with the first Ndct coefficients x is the signal Ndct is the number of DCT coefficients to be calculated sr is the sampling rate of the signal

Parameters:

  • x – Signal to be analyzed.

  • Ndct – Number of DCT coefficients to be calculated.

  • sr – Sampling rate.

Returns:

  • coefs: DCT coefficients.

  • approx: Reconstruction of X using the Ndct number of DCT coefficients.
