alignment.get_vals¶

alignment.get_vals(filename, midi_file, audiofile, sr, hop, width, target_sr, nharm, win_ms)[source]¶

Gets values for DTW alignment and YIN analysis of specified audio signal and MIDI file

Parameters:

  • filename – Name of the audio file.

  • midiname – Name of the MIDI file.

  • audiofile – Name of teh audio file.

  • sr – Sample rate.

  • hop – Hop size.

Returns:

  • res.on list of DTW predicted onset times in seconds

  • res.off list of DTW predicted offset times in seconds

  • yinres.ap aperiodicty estimates for each frame

  • yinres.pwr power estimates for each frame
